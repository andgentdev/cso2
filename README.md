# CSO2 (v2.2) — Transparent PNG Overlay for Windows

CSO2 is a lightweight Windows utility that displays PNG images as transparent, always-on-top overlays.
It is designed for quick visual reference, graphic comparison, and desktop overlay workflows.

## Download (recommended)

If you just want to use CSO2 without installing Python, download the prebuilt executable from **GitHub Releases**:

- Go to the repository page → **Releases**
- Download the latest `cso2.2.exe` (or the provided ZIP, if available)

## Features

- True PNG transparency support
- Frameless, always-on-top overlay windows
- Mouse wheel zoom with smooth scaling
- Horizontal mirror toggle via keyboard arrows (Left/Right)
- Click-and-drag window positioning
- Right-click or `ESC` to close a window
- Multiple independent overlays per session

## Usage

### Run

- Drag and drop a PNG file onto the executable, or
- Run from command line:

```bash
cso2.2.exe "C:\path\to\image.png"
